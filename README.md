 💬 Real-Time Chat Application (Group + Private Messaging)

🛠️ Tech Stack

**💻 Frontend**

React.js – UI development using function components and hooks (useState, useEffect, useRef)

Tailwind CSS – For utility-first responsive styling

Socket.IO Client – Real-time communication via WebSockets (socket.io-client)

Axios – HTTP client for API requests

Emoji Picker React – Emoji selection component

React Icons – Icon library (react-icons, lucide-react)

**🌐 Backend**

Node.js – JavaScript runtime

Express.js – Web framework for building RESTful APIs

Socket.IO – Real-time bidirectional communication

Mongoose – MongoDB object modeling tool for Node.js

MongoDB – NoSQL database for message and user data storage

**Security & Auth**

- **JWT** for authentication

- **bcrypt.js** for password hashing
